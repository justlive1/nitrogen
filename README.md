# nitrogen

## 介绍
基于vue的纯前端mac版桌面

![效果图](https://gitee.com/justlive1/nitrogen/raw/master/public/images/snapshot.jpg)

## 快速开始

安装node环境

```
// 下载依赖
npm i
// 启动开发环境
npm run dev
// 构建
npm run build
```

安装部署

- 单独使用，将 `dist` 文件夹下内容放到 http服务器对应目录 （nginx/apache），访问对应地址即可
- 放到项目中使用，将 `dist` 文件夹下内容拷贝到项目静态资源路径下或自行修改对应资源和页面
